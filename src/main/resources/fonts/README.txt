本目录字体用于 MathAnim 桌面端 UI。

- Inter-Variable.ttf：SIL Open Font License 1.1，见 Inter-OFL.txt（来自 google/fonts ofl/inter）。
- JetBrainsMono-Regular.ttf：JetBrains Mono，SIL OFL 1.1，见 https://github.com/JetBrains/JetBrainsMono
- LXGWWenKaiGBScreen.ttf：霞鹜文楷屏幕优化版（GB），SIL OFL 1.1，来源 https://github.com/lxgw/LxgwWenKai-Screen ，与 ManimCat Web 使用同款屏显文楷思路；正文书写感优先使用，全大写标签与按钮仍用 Inter。
